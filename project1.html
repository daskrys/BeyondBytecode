GreenBot Rover

This project was part of the NASA California Space Grant Consortium 2019 done at Orange Coast College and the timeline was 2 months. I worked on a team to design and create an Arduino based project. Program concluded with a presentation at NASA Armstrong Flight Research Center. I was the programmer for this project. Our rover measured light intensity, soil moisture, & temperature. Output was sent via bluetooth to an application on our Phone.

View On GitHub
Description

GreenBot is a Rover designed for agriculture using a Vex + Arduino system that transmits data from multiple sensors to a phone application using Blynk IoT.

Arduino: Soil Moisture Capacitive Sensor, DSD BLE SH-08 Bluetooth Module, DHT11 Humidity + Temperature Sensor, Photoresistor

Vex: 5 Servo Motors, 3 Shaft Encoders, Blynk displays data from Arduino system and outputs it onto Blynk app through bluetooth All of the code for this project was done using C/C++

Team: Christian Perez, Mohamed Ismail, Sarah Catania, Shwetank Bhat
